Entenda a API

  • Ler Profiles (função read)
    http://localhost/crud/?fn=read.
  • Criar Profiles (função create + first_name + last_name + dbo + email)
    http://localhost/crud/?fn=create&first_name=Mary&last_name=Ann&dbo=30121999&gender=F&email=matheus@mail.com
  • Atualizar Profiles (função update + first_name + last_name + dbo + email)
    http://localhost/crud/?fn=update&first_name=Matheusinho&last_name=Araujo&dbo=29121999&gender=M&email=matheusinho@mail.com&id=1
  • Deletar Profiles (função delete + id)
    http://localhost/crud/?fn=delete&id=1
  • Criar Report (função createReport + first_nameReport + id [do Report a ser adicionado])
    http://localhost/crud/?fn=createReport&nameReport=OutroReport&fk_id=1
  • Ver Reports (função readReport + titleReport)
    http://localhost/crud/?fn=readReport

Outras funcionalidades da API

  • É possível consultar por id do Profile:
    http://localhost/crud/?fn=read&id=2
  • É possível consultar Reports pelo id do Report:
    http://localhost/crud/?fn=readReport&id=5

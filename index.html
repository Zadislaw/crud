Entenda a API

  • Ler Contatos (função read)
    http://localhost/crud/?fn=read.
  • Criar Contatos (função create + nome + sobrenome + dtnascimento + telefone + celular + email)
    http://localhost/crud/?fn=create&nome=Matheus&sobrenome=Costa&dtnascimento=30121999&telefone=2125554444&celular=21955554444&email=matheus@mail.com
  • Atualizar Contatos (função update + nome + sobrenome + dtnascimento + telefone + celular + email)
    http://localhost/crud/?fn=update&nome=Matheusinho&sobrenome=Araujo&dtnascimento=29121999&telefone=8225554444&celular=82955554444&email=matheusinho@mail.com&id=1
  • Deletar Contatos (função delete + id)
    http://localhost/crud/?fn=delete&id=1
  • Criar empresa (função createEmp + nomeEmp + id [do contato a ser adicionado])
    http://localhost/crud/?fn=createEmp&nomeEmp=PrimeiraEmpresaGRANDE&fk_id=1
  • Ver empresas (função readEmp + nomeEmp)
    http://localhost/crud/?fn=readEmp

Outras funcionalidades da API

  • É possível consultar por id da pessoa:
    http://localhost/crud/?fn=read&id=2
  • É possível consultar empresas pelo id dos contatos:
    http://localhost/crud/?fn=readEmp&id=5
